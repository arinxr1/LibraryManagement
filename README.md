# 📚 Library Management System

A simple relational database project built using MySQL.

## Project Description

This project is designed to manage an electronic library by storing and organizing:

- Authors
- Books
- Members
- Borrowing records

The system allows:

- Register new authors
- Register new books
- Register members
- Borrow books
- Return books
- Track book availability

---

## Database Tables

- Authors
- Books
- Members
- Borrowings

---

## Technologies

- MySQL
- MySQL Workbench

---

## Database Design

- Primary Keys
- Foreign Keys
- Indexes
- AUTO_INCREMENT
- Relationships (One-to-Many)
